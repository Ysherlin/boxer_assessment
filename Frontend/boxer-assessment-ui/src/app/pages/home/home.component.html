Employee Registry

Welcome to the Boxer Assessment Employee Registry. This application allows you to manage employees, search records, and maintain up-to-date employee information.

About

This system is built using Angular and .NET 8 as part of a technical assessment. It demonstrates clean architecture, RESTful APIs, and a simple but effective user interface.
